Model the following Doctor’s information system as a graph model, and
answer the following queries using Cypher.
Consider the doctors in and around Pune. Each Doctor is specialized in
some stream like Pediatric, Gynaec, Heart Specialist, Cancer Specialist,
ENT, etc. A doctor may be a visiting doctor across many hospitals or he
may own a clinic. A person can provide a review/can recommend a
doctor.
1. Identify the labels and relationships, along with their properties, and
draw a high-level Graph model.  
2. Create nodes and relationships, along with their properties, and
visualize your actual Graph model.  
3. Answer the Queries
a. List the Orthopedic doctors in ……. Area.  
b. List the doctors who has specialization in ____  
c. List the most recommended Pediatrics in Seren Medows. 
d. List all the who visits more than 2 hospitals

        CREATE (:Hospital {name: "Seren Medows Hospital", area: "Seren Medows"}),
               (:Hospital {name: "Kothrud Clinic", area: "Kothrud"}),

               (:Hospital {name: "Baner Health Center", area: "Baner"});
        `;

               (:Person {name: "Suresh", review: "Good", recommendation_count: 3}),
               (:Person {name: "Maya", review: "Outstanding", recommendation_count: 7});
        `;

        // Relationships
        const createRelationships = `
        MATCH (d:Doctor {name: "Dr. Patel"}), (h:Hospital {name: "Kothrud Clinic"})
        CREATE (d)-[:VISITS]->(h);

        CREATE (d)-[:VISITS]->(h);

        MATCH (p:Person {name: "Ramesh"}), (d:Doctor {name: "Dr. Joshi"})
        CREATE (p)-[:RECOMMENDED]->(d);

        MATCH (p:Person {name: "Maya"}), (d:Doctor {name: "Dr. Joshi"})
        CREATE (p)-[:RECOMMENDED]->(d);

        MATCH (p:Person {name: "Suresh"}), (d:Doctor {name: "Dr. Patel"})
        CREATE (p)-[:RECOMMENDED]->(d);
        `;

        // a) List the Orthopedic doctors in a specific area
        const queryOrthopedicDoctors = `
        MATCH (d:Doctor {specialization: "Orthopedic", area: "Kothrud"})
        RETURN d.name AS DoctorName, d.area AS Area;
        `;

        // b) List the doctors who have a specialization in a given field
        const queryDoctorsBySpecialization = `
        MATCH (d:Doctor {specialization: "Heart Specialist"})
        RETURN d.name AS DoctorName, d.specialization AS Specialization;
        `;

        // c) List the most recommended Pediatric in Seren Medows
        const queryMostRecommendedPediatric = `
        MATCH (d:Doctor {specialization: "Pediatric", area: "Seren Medows"})<-[:RECOMMENDED]-(p:Person)
        RETURN d.name AS DoctorName, COUNT(p) AS Recommendations
        ORDER BY Recommendations DESC
        LIMIT 1;
        `;

        // d) List all doctors who visit more than 2 hospitals
        const queryDoctorsMoreThanTwoHospitals = `
        MATCH (d:Doctor)-[:VISITS]->(h:Hospital)
        WITH d, COUNT(h) AS HospitalCount
        WHERE HospitalCount > 2
        RETURN d.name AS DoctorName, HospitalCount;
        `;
