 the queries using Cypher. Consider a car showroom with different models of
 cars like sofas Honda city, Skoda, Creta, Swift, Ertiga etc. Showroom is
 divided into different sections, one section for each car model; each section is
 handled by a sales staff. A sales staff can handle one or more sections.
 Customer may enquire about car. An enquiry may result in a purchase by the
 customer.
1. Identify the labels and relationships, along with their properties, and
draw a high-level Graph model.  
2. Create nodes and relationships, along with their properties, and
visualize your actual Graph model.  
3. Answer the following queries:
a. List the types of cars available in the showroom.  
b. List the sections handled by Mr. Narayan.  
c. List the names of customers who have done only enquiry but not
made any purchase.  
d. List the highly sale car model.

    <!-- Neo4j Graph Model and Cypher Queries -->
    <script>
        // Neo4j Graph Model and Cypher Queries for Car Showroom Management

        const createNodes = `
        // Create Car Models
        CREATE (:Car {model: "Honda City", type: "Sedan", price: 1200000}),
               (:Car {model: "Skoda", type: "Sedan", price: 1500000}),
               (:Car {model: "Creta", type: "SUV", price: 1800000}),
               (:Car {model: "Swift", type: "Hatchback", price: 800000}),
               (:Car {model: "Ertiga", type: "MPV", price: 900000});

        // Create Sections
        CREATE (:Section {name: "Honda City Section"}),
               (:Section {name: "Skoda Section"}),
               (:Section {name: "Creta Section"}),
               (:Section {name: "Swift Section"}),
               (:Section {name: "Ertiga Section"});

        // Create Sales Staff
        CREATE (:SalesStaff {name: "Mr. Narayan", experience: "5 years"}),
               (:SalesStaff {name: "Ms. Singh", experience: "3 years"});

        // Create Customers
        CREATE (:Customer {name: "John Doe", enquiry_date: "2023-11-01", purchase_date: null}),
               (:Customer {name: "Jane Smith", enquiry_date: "2023-11-02", purchase_date: "2023-11-03"});

        // Create Relationships
        MATCH (c:Car {model: "Honda City"}), (s:Section {name: "Honda City Section"})
        CREATE (c)-[:AVAILABLE_IN]->(s);

        MATCH (c:Car {model: "Skoda"}), (s:Section {name: "Skoda Section"})
        CREATE (c)-[:AVAILABLE_IN]->(s);

        MATCH (c:Car {model: "Creta"}), (s:Section {name: "Creta Section"})
        CREATE (c)-[:AVAILABLE_IN]->(s);

        MATCH (c:Car {model: "Swift"}), (s:Section {name: "Swift Section"})
        CREATE (c)-[:AVAILABLE_IN]->(s);

        MATCH (c:Car {model: "Ertiga"}), (s:Section {name: "Ertiga Section"})
        CREATE (c)-[:AVAILABLE_IN]->(s);

        MATCH (staff:SalesStaff {name: "Mr. Narayan"}), (sec:Section {name: "Honda City Section"})
        CREATE (staff)-[:HANDLES]->(sec);

        MATCH (staff:SalesStaff {name: "Mr. Narayan"}), (sec:Section {name: "Skoda Section"})
        CREATE (staff)-[:HANDLES]->(sec);

        MATCH (staff:SalesStaff {name: "Ms. Singh"}), (sec:Section {name: "Creta Section"})
        CREATE (staff)-[:HANDLES]->(sec);

        MATCH (customer:Customer {name: "John Doe"}), (car:Car {model: "Swift"})
        CREATE (customer)-[:ENQUIRED]->(car);

        MATCH (customer:Customer {name: "Jane Smith"}), (car:Car {model: "Swift"})
        CREATE (customer)-[:PURCHASED]->(car);
        `;

        const queries = {
            listCarTypes: `
            MATCH (c:Car)
            RETURN DISTINCT c.type AS CarType;
            `,
            sectionsHandledByNarayan: `
            MATCH (s:SalesStaff {name: "Mr. Narayan"})-[:HANDLES]->(sec:Section)
            RETURN sec.name AS SectionName;
            `,
            enquiryButNoPurchase: `
            MATCH (c:Customer)-[:ENQUIRED]->(car:Car)
            WHERE c.purchase_date IS NULL
            RETURN c.name AS CustomerName;
            `,
            highlySoldCarModel: `
            MATCH (c:Car)<-[:PURCHASED]-(customer:Customer)
            RETURN c.model AS CarModel, COUNT(customer) AS Sales
            ORDER BY Sales DESC
            LIMIT 1;
            `,
        };

        console.log('Nodes Creation:');
        console.log(createNodes);

        console.log('Queries:');
        console.log('a) List of car types available in the showroom:', queries.listCarTypes);
        console.log('b) Sections handled by Mr. Narayan:', queries.sectionsHandledByNarayan);
        console.log('c) Customers who enquired but didn\'t make any purchase:', queries.enquiryButNoPurchase);
        console.log('d) Highly sold car model:', queries.highlySoldCarModel);
    </script>
